# 💰 Expense Tracker

A modern and responsive **Expense Tracker** web application built using **HTML, CSS, and JavaScript**. This application helps users manage their income and expenses, calculate the current balance, and securely store transaction data using the browser's Local Storage.

---

## ✨ Features

* ➕ Add Income Transactions
* ➖ Add Expense Transactions
* 💰 View Current Balance
* 📊 Display Total Income & Total Expenses
* 🗑️ Delete Transactions
* 💾 Store Data Using Local Storage
* 📱 Responsive User Interface

---

## 🛠️ Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript (ES6)
* 💾 Local Storage

## 🚀 Getting Started

1. Clone or download this repository.
2. Open the project folder.
3. Open **index.html** in your web browser.

Or

* Open the project in **Visual Studio Code**.
* Use the **Live Server** extension to run the application.
